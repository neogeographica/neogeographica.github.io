Omicron Fixup

  • AI deathmatch "bot" mod for Quake (not QuakeWorld)
  • fix minor annoyances in the original Omicron Bots mod
  • released August 2020

Another little tweak here for a grand old Quake bots mod. Omicron Bots are still a good choice for Quake bots when it comes to DM maps that don't have any explicit waypoint support for other bots mods, as well as being a fun knockaround time in their rocket arena mode. As was the case with FrikBot X+ however, the existing package had just a few rough edges that I've smoothed out. This is a really tiny set of changes but you might find them nice to have!
