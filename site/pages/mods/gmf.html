Ghosts Made Flesh

  • "tourism" mod for Quake (not QuakeWorld)
  • a patch for the "Ghosts" mod
  • released April 2018

This is a patch/update for JP LeBreton's Ghosts mod, a "tourism" mod that makes several changes to allow safely wandering through the levels of Quake. It also changes the soundtrack to use the Nine Inch Nails album "Ghosts I-IV".

The original version of Ghosts removes all monsters from the game. This tweaked version of Ghosts restores the monsters, although they will not attack the player. You might like this variant if you feel that the presence of the Quake monsters is part of the ambience of the levels.

You will need to download and install the original Ghosts mod before applying this patch. More details in the readme.
