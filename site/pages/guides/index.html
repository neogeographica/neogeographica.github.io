A few things to note up front:

  • These guides are all for Quake... not Quake 2 or Quake 3.
  • Originally these were posted to the Steam community content for Quake. The guide pages on this site are an automated conversion from Steam's BBCode markup.
  • The vast majority of the info in these guides applies to any Quake installation, not only the stuff you get from Steam. I just added a few Steam-specific tips to help those folks along.

These guides should still be on-point for handling the original Quake engines. But when it comes to the specific features of modern Quake engines, I've let these guides get out of date. There's some errata posted over on the Steam forums but that may be getting stale at this point too.

I'm not sure when/if I'll be in the frame of mind for a grand rewrite of this stuff. My "help folks play Quake" energy is entirely taken up by the Quake Singleplayer Starter Pack currently, which is a project that is a bit more focussed and practical.

Regardless, if you've got comments about the guide contents you can still send me an email, especially if you see some info about the original Quake engines that needs correcting.
